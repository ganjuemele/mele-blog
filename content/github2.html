使用github搭建静态页面

进入 GitHub 网站新建项目，选取 ssh地址

  1. 新建 project 文件夹

    mkdir project

  2. 进入 project

    cd project

  3. 新建 页面

    touch index.html

  4. 使用编辑器在 index.html 文件中写入你要展示的页面

  5. 本地打开终端,进入文件夹

    初始化git

    git init

  6. 添加需要git的目录

    git add .

  7. 提交add的文件版本

    git commit -v

  8. 添加远程仓库地址

    git remote add origin (github项目的ssh地址)

  9. 上传master分支到 远程仓库

    git push -u origin master
  10. 进入github项目 >> setting >> GitHub Pages >> 选择 master 分支
    静态页面 搭建成功
